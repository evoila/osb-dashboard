Backup

Based on the OSB Backup Manager Framework this Service Broker offers the ability to create a fully feature Backup schedule for your Service Instance. Currently we support the following features:

  • File Endpoints
  • Backup Plans
  • Restore Points
  • Backup Jobs
  • Restore Jobs
Configure Backup
Notification

You can configure notifications which will trigger on scheduled backups, so that E-Mails will be sent on certain events (SUCCESS, FAILED, etc.) with information about the Service Broker and why the event happened.

Configure Notification
Management

The Management section offers functionality which enables extended functionality. In case of database based Service Brokers like PostgreSQL, MySQL or MongoDB it contains functionality to create new database for Dedicated Service Instances.

Service Keys

Service Keys allow you to access your Service Instance over a persistent IP and Port from the outside. We strongly recommend to NOT use this feature, because it a large security risk. (IMPORTANT: This functionality is only useable in Cloud Foundry Environments.)
